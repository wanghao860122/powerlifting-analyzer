# 健力課表分析器 · Powerlifting Progress Analyzer

> 把教練的 Google Sheets 課表貼上來，自動分析三大項的 RPE 進步趨勢、量化進步率與 SBD 總和 e1RM。

**[立即使用 →](https://wanghao860122.github.io/powerlifting-analyzer/)**

---

## 隱私聲明

**你的課表資料絕對不會離開你的裝置。**

本工具完全在瀏覽器本地端運算：
- 不會上傳任何資料到伺服器
- 不會儲存你的 Google Sheets 連結或訓練數據
- 僅透過 Google 官方的 gviz API 公開讀取你分享的試算表（唯讀）
- 關掉分頁即清除所有暫存

---

## 快速開始

1. 將你的 Google Sheets 課表設為「**知道連結的任何人均可檢視**」  
   （試算表右上角 → 共用 → 變更 → 知道連結的任何人 → 檢視者）
2. 複製網址
3. 開啟本工具，貼上網址，填入體重與性別，點「**分析**」

## Google Sheets 格式要求

> 本工具設計對應特定的課表格式，若格式不符則無法正確讀取。

### 分頁命名

課表必須以 **`Cycle1`、`Cycle2`、`Cycle3`……** 命名分頁，工具會自動依序掃描所有 Cycle。

```
試算表分頁標籤：
┌──────────┬──────────┬──────────┬─────────┐
│  Cycle1  │  Cycle2  │  Cycle3  │ (其他)  │
└──────────┴──────────┴──────────┴─────────┘
                                    ↑ 不含「Cycle」開頭的分頁會被自動略過
```

### 欄位結構（每個 Cycle 分頁）

每個週次佔 **5 欄**，週次標題放在第 1 列，第 2 列為欄位標題，第 3 列起為訓練資料：

```
第 1 列 → 週次標題（如 "W1 2024/01/08"）
           ↓
         ┌────────────┬──────────┬────────────┬──────────┬──────────┐
第 2 列  │  exercise  │  volume  │ intensity  │  weight  │   RPE    │
         ├────────────┼──────────┼────────────┼──────────┼──────────┤
第 3 列  │  Squat     │  3x5     │   Rpe7     │   120    │    7     │
第 4 列  │  Bench     │  4x4     │   Rpe6.5   │    80    │    7     │
第 5 列  │  Deadlift  │  3x5     │   Rpe7.5   │   150    │    8     │
第 6 列  │  ...       │  ...     │   ...      │   ...    │   ...    │
         └────────────┴──────────┴────────────┴──────────┴──────────┘
```

| 欄位 | 說明 | 格式範例 |
|------|------|----------|
| **exercise** | 動作名稱 | `Squat`、`Bench Press`、`Deadlift` |
| **volume** | 組數 × 次數 | `3x5`、`3*5`、`4x3` |
| **intensity** | 教練計畫 RPE（選填，用於爆掉偵測） | `Rpe6`、`RPE 7.5`、`rpe8` |
| **weight** | 實際使用重量（公斤，數字） | `120`、`82.5` |
| **RPE** | 實際感受費力程度（數字，1–10） | `7`、`8.5` |

### 動作名稱辨識規則

工具以**關鍵字比對**識別三大項，不限制語言或全名：

| 動作 | 可辨識的關鍵字 |
|------|---------------|
| 深蹲 | 名稱含 `squat`（不分大小寫） |
| 臥推 | 名稱含 `bench` |
| 硬舉 | 名稱含 `deadlift` 或 `dead` |

- `Low Bar Squat`、`Pause Squat`、`Comp Squat` → 全算深蹲 ✓
- `Bench Press`、`Paused Bench`、`Close Grip Bench` → 全算臥推 ✓
- `Conventional Deadlift`、`Sumo Dead` → 全算硬舉 ✓
- `Romanian Deadlift (RDL)`、`Dead Bug`、`Leg Press`、`Dumbbell` 等輔助動作 → 自動排除 ✗

## 功能介紹

### SBD 總和 e1RM 趨勢
使用 RTS RPE 百分比表，將每次訓練換算為估計最大重量（e1RM），並追蹤三大項合計的進步曲線。

### 各項 RPE 折線圖
每個 RPE 對應一條彩色折線，呈現固定費力程度下的重量變化——可直觀看出在哪個 RPE 的進步最穩定。可點選下方 RPE 按鈕隱藏/顯示特定線條。

### 量化進步率
對每個 RPE 的重量做線性迴歸，計算「每週進步幾公斤」，排除努力程度變化的干擾。

### RPE 爆掉偵測
比較課表的計畫 RPE（Intensity 欄）與實際 RPE，記錄超出計畫的訓練組，統計各項的爆掉頻率。

### Wilks / DOTS 係數
輸入體重與性別後，自動換算成比賽標準係數，追蹤不同體重級別下的相對實力。

### 多 Cycle 支援
自動載入所有 Cycle 分頁，可選擇分析特定 Cycle 或自訂週數區間。
